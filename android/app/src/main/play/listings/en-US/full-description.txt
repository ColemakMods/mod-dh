This tiny app allows you to connect a physical keyboard to your device using USB or Bluetooth, and then type using one of the Colemak Mod-DH layout mappings.

<b>Supported layouts:</b>
<ul>
	<li>Mod-DH ANSI US</li>
	<li>Mod-DH ANSI US wide</li>
	<li>Mod-DH ISO US</li>
	<li>Mod-DH ISO US wide</li>
	<li>Mod-DH ISO UK</li>
	<li>Mod-DH ISO UK wide</li>
	<li>Vanilla Colemak</li>
	<li>Vanilla Colemak wide</li>
</ul>

<b>Note:</b>
<ul><li>This app is for physically connected keyboards only - it does <b>NOT</b> change the on-screen software keyboard.</li></ul>
